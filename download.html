Download OpenSees for Fire

Download Exectuable for Windows

The current version is based on OpenSees 2.4.0, and already includes 2D and 3D beam-column elements and shell elements for modelling framed structures. SIFBuilder project has been also packed in the executable file. To use it, please refer to the latest user manual

Download OpenSeesforFire.exe

Download Tcl/TK

OpenSees uses the Tcl interpreter which has been extended with model builder commands. Before using OpenSees, the Tcl library should be installed and the installation file can be found from this site:

Activate Tcl Website

It should be noticed that the Tcl/tk is recommanded to be installed into C:/Program files/Tcl (NOTE: The current version only supports the 64bit version of Tcl. 32bit version has to be compiled by the user himself)


Browse Source Code

Source code on GitHub

Source code on Subversion

You can also use any SVN clients to download the source code package.. Then you can compile it and build it on your own desktop with a proper development tool such as Visual Studio.

SVN URL:https://svn.ecdf.ed.ac.uk/repo/see/OpenSeesEd/OpenSees/

Development Environment

This project is developed using Microsoft Visual C++ 2008, the most recommended IDE would be Microsoft Visual Studio 2008 (or higher version). But please do pay your attention to the proj file which may cause problems
